三、η 比較區 Distribution Index Comparison

直接比較小 η 與大 η 的分布差異。左側較敢跳，右側較保守。

Small η / 較強探索

η = 2

當 η 較小時，樣本分布較寬，較容易出現遠離 parent 或原值的位置。
When η is small, the distribution spreads out more, encouraging exploration.
Large η / 較強局部搜尋

η = 35

當 η 較大時，樣本分布較集中，更偏向在 parent 或原值附近做局部搜尋。
When η is large, the distribution becomes concentrated and more exploitative.

五、重點整理 Key Takeaways

這個頁面的目的，不是背公式，而是看懂 operator 的行為。

SBX
父代導向的 crossover。它不一定只在父代中間產生 offspring。
Polynomial Mutation
單點導向的擾動。靠近邊界時，分布可能不對稱。
η 的角色
η 小通常探索較強；η 大通常比較保守，偏向局部搜尋。
1D 優先
1D 最適合教學；2D 是幫助理解多維搜索行為的補充圖像。
建議課堂操作方式：先讓學生在 1D 中調整 η 與 parent 距離，再觀察 2D cloud，最後請學生用自己的話說出 exploration / exploitation 的差別。
