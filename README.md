# metaheuristics-gadget
Small interactive tools for learning metaheuristics

| Gadget  | Descriptions |
| ----- | -------- |
| [Metaheuristics Explorer](https://tcchiang.github.io/metaheuristics-gadgets/metaheuristics-explorer.html) | II vs. SA vs. TS |
| [Mating Selection Explorer](https://tcchiang.github.io/metaheuristics-gadgets/mating-selection-explorer.html) | Visualization of six mating selection variants |
| [SBX & PM Visualizer](https://tcchiang.github.io/metaheuristics-gadgets/sbx-polynomial-mutation-vis.html) | SBX & PM visualizer |
| [GA for TSP](https://tcchiang.github.io/metaheuristics-gadgets/tsp-ga-explorer.html) | A simple demo of using GA to solve TSP |
| [Permutation Crossover Visualizer](https://tcchiang.github.io/metaheuristics-gadgets/permutation-encoding-crossover-visualizer.html) | Five common crossover for permutation encoding |
| [Lamarckian vs. Baldwinian](https://tcchiang.github.io/metaheuristics-gadgets/baldwin-lamarck-visualizer.html) | Memetic algorithm: Lamarckian vs. Baldwinian |
| [EMO: Scalarization Explorer](https://tcchiang.github.io/metaheuristics-gadgets/scalarization-explorer-v2.html) | WS vs. TCH vs. PBI |
